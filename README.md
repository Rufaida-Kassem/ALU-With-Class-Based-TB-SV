# ALU-With-Class-Based-TB-SV
A verification environment for an ALU using class-based testbench in `Systemverilog`
The focus is on the *verification environment*

#### Supported ALU operations:
- NOP
- ADD
- SUB
- MUL
- DIV
- SL
- SR
- AND
- OR
- NOT
- XOR
